Digit span memory tests: how many numbers can you remember?

Each sequence will be displayed for 1/2 second per digit plus 1 second
- you start with 4 digits
- every correct answer the digit length increases by 1
- every incorrect answer the digit length decreases by 1

digit span using NUMBERS (10 different digits)

digit span using LETTERS (upper-case only) (26 different digits)

digit span using LETTERS (upper and lower case) (52 different digits)

digit span using upper or lower case letters or numbers (62 different digits)

BEWARE: make sure you clearly identify 0 or o or O! And 1 or I or l or i!
