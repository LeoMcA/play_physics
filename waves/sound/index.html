Speed of Sound Experiments simulations:

Using 2 microphones:

Position 2 microphones, create a 'bang', and the time taken for the sound to travel between the microphones is recorded. Do it a number of times and you'll be able to plot a distance vs. time graph to find the speed of sound

Using 1 microphone and an echo

Position the distance between the microphones and the wall, create a 'bang', and the time taken for the sound to travel from the microphone to the wall, and back again, is recorded. Do it a number of times and you'll be able to plot a distance vs. time graph (either distance or time will need recalculating...) to find the speed of sound

Using 1 microphone and an echo - quick version

The same Sim as the previous one but this one gives an instant read-out of the calculated speed using the measured data.
